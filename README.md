### Self-serve시 유의사항

`css.local` 디렉토리의 파일을 이용해 주세요.

### Apple SD Gothic Neo

**_HTML_**

```
<link rel="stylesheet" href="//cdn.jsdelivr.net/npm/font-applesdgothicneo@1.0/all.min.css">
```

**_CSS_**

```
@import "//cdn.jsdelivr.net/npm/font-applesdgothicneo@1.0/all.min.css";
```
